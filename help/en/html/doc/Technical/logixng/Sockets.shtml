LogixNG - Sockets

Sockets

LogixNG is designed to be flexible and easy to extend. A major idea is that tools should not know details of expressions and actions.

  • ITEM
    An item on the layout, for example turnout, sensor or signal mast.
  • COMMON
    Commonly used items. For example ExpressionAnd, ExpressionOr, ExpressionTimer, ActionMany and ActionDoIf.
  • OTHER
    All other expressions and actions which don't fit in the above categories.
